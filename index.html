GAME SHOW BUZZER

TECHNOLOGY

  • HTML
  • CSS
  • JQuery
  • Node.js
  • Express
  • Websocket.io

This game show buzzer system allows users to join the session via their phone to buzz in. there are unlimited teams, with unlimited players in each team.

The leaderboard keeps track of the teams scores, and shows who buzzed in.

The player screen shows the teams name and score. Each team can have their own image as a background.

The system also includes a webpage for incorporation into PowerPoint.

EVENT PRODUCTION TIMER

TECHNOLOGY

  • HTML
  • CSS
  • JQuery
  • Node.js
  • Express
  • Websocket.io

The client requested a web based timer system that would allow a session chair to control the timer from side of stage.

The system allows multiple timers for multiple rooms and includes a main monitoring screen where event organisers can monitor sessions.

Each timer screen can have it's CSS modified via the controll screen, including uploading of background images, and control of the timer can be transfered between devices.

CHEAP DATE - YOUR AT HOME BAR

TECHNOLOGY

  • HTML
  • CSS
  • Bootstrap
  • JQuery

We are a bunch of university students currently learning coding! this website was built as one of our first projects, where we were given the task of incorporating the skills we've learnt throughout the past 6 weeks.

We wanted to create something that was fun and easy to use, while informative at the same time. we realised that the vast majority of people who enjoy cocktails usually drink them when out at a restaurant or bar. but what if someone doesn't feel like going out just to enjoy their favourite cocktail? that's where we come in.

We allow users to search for their favourite cocktail, browse the numerous recipes we have to offer, or create their own with the alcohol they already have! we not only display the recipe and required ingredients, but also the average price the user would pay if they were to go out and buy everything they dont already have.
